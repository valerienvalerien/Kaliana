# Kaliana

Page d'accueil de **Kaliana**, maison de couture haut de gamme pour enfants basée à Antananarivo, Madagascar.

L'atelier réalise à la main des robes smockées, brodées et personnalisées dans la tradition des grandes maisons de couture malgaches.

## Aperçu

- Design léger et aéré, dégradé rose poudré autour de `#FF69B4`.
- Typographie sans-serif (Inter) + serif italique (Playfair Display) pour les accents éditoriaux.
- Sections : navigation, hero, valeurs, collection (3 modèles), savoir-faire, services, devis, pied de page.
- 100 % statique : HTML + CSS, aucune dépendance JS.

## Lancer en local

Ouvrir simplement `index.html` dans un navigateur, ou servir le dossier :

```bash
python3 -m http.server 8000
```

Puis visiter http://localhost:8000.

## Fichiers

- `index.html` — Structure et contenu de la page.
- `styles.css` — Système de design, palette rose, mises en page.
